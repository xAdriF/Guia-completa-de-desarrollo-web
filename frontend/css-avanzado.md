# CSS Avanzado

## Introducción
El **CSS avanzado** incluye herramientas modernas que permiten crear sitios web más flexibles, responsivos y accesibles mediante nuevos selectores, funciones y sistemas de diseño.

## Conceptos Principales

### Selectores Modernos
- `:is()` para simplificar listas.
- `:has()` para aplicar estilos según elementos hijos.
- Selectores de atributo para casos específicos.

### Diseño y Responsividad
- **Grid** para estructuras complejas.
- **Flexbox** para organizar elementos.
- Funciones como `clamp()` y unidades como `dvh` para tamaños adaptables.

### Variables y Accesibilidad
- Variables CSS para colores y espaciados.
- Modo oscuro con `prefers-color-scheme`.
- Reducción de animaciones con `prefers-reduced-motion`.
