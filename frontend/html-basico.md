# HTML Básico

## Introducción
**HTML (HyperText Markup Language)** es el lenguaje estándar para estructurar el contenido de las páginas web. Define elementos como texto, imágenes, enlaces y secciones mediante etiquetas.

## Elementos Fundamentales

### Estructura Básica de una Página
```html
<!DOCTYPE html>
<html>
  <head>
    <title>Mi página</title>
  </head>
  <body>
    <h1>Título</h1>
    <p>Texto de ejemplo.</p>
  </body>
</html>
